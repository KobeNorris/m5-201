# 前端开发规范

## 格式规范
列举几个格式规范的例子：
- 等号两边要加括号
- if 和 for 后面的括号前要空格
- 句子结尾不能带分号


不存在最好的格式规范。每个开发团队可以有不同的规范。Leader 们可以要求组员在 `HBuilder` 中下载 `eslint`，然后配置你们需要的规范（或者使用默认规范）。这样，`HBuilder` 就会在规范不正确的时候报错。
