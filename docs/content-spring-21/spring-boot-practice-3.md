# SpringBoot 练习 3

## 基础
根据本周介绍的 `后端开发规范`，优化之前两个 `SpringBoot` 练习中的实现的接口，要求：  
1. 代码风格统一。  
2. 使用 `Swagger` 生成接口文档。  
3. 使用 `Service` 分担 `Controller` 的工作，方便复用。  
4. 使用 `CommonResult` 封装返回信息。  
5. 使用 `CommonResult` 以及 `Exception` 处理好可能的错误情况。  


## 进阶
为自己正在进行的项目进行数据库设计（如果已经有数据表了，可以跳过），然后制作对应的后端接口。
