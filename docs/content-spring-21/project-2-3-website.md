**注意：与剩余所有的项目不同，这个项目没有后台部分。**这意味着，你无法在 `Project 3` 中继续开发它的后台，只能加入其它队伍，开发其它项目的后台。  

我们提供了专业的设计稿（把每个参数都详细定义了），可以带给大家最最接近真实的企业前端开发的体验。
