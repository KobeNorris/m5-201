# Web 练习

## 准备工作
- 请在桌面上创建一个文件 `myWebPage.html`。
- 右键，使用 `VSCode` 或者 `WebStorm` 打开。
- 双击，使用浏览器打开。
- 把编辑器（代码）放在左边，网页放在右边。这样，每当你改变代码并保存，刷新网页就会自动更新。
